# Founder Growth Agent

AI-powered founder-led B2B growth strategist for solo founders and early-stage teams. Positioning, messaging, content, and outreach strategies — organic, $0-budget B2B growth.

## Features

- **Founder-led growth strategy**: Positioning, messaging, content, outreach
- **Organic pipeline generation**: LinkedIn, X/Twitter, cold outreach, partnerships
- **Zero-budget focus**: Tactics that work with little to no marketing spend
- **Real-time streaming**: Instant responses as they generate
- **Structured outputs**: Clear frameworks, action plans, and templates

## Tech Stack

- **Next.js 16** — React framework
- **Tailwind CSS v4** — Utility-first styling
- **OpenAI SDK** — GPT-4o streaming
- **TypeScript** — Type safety
- **Vercel** — Deployment

## Getting Started

### 1. Clone and install

```bash
git clone https://github.com/21leahcimhtiek-oss/founder-growth-agent.git
cd founder-growth-agent
npm install
```

### 2. Set environment variable

```bash
cp .env.example .env.local
# Add your OpenAI API key:
OPENAI_API_KEY=sk-...
```

### 3. Run dev server

```bash
npm run dev
```

Open http://localhost:3000

## Deployment

This repo is set up for Vercel deployment. Connect the repo on Vercel and add `OPENAI_API_KEY` as an environment variable.
